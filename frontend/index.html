Platform Features

Everything You Need for
Mineral Exploration

From interactive GIS mapping to AI-powered predictions and professional PDF reporting — all in one platform.

Interactive GIS Map

Click to select any location in Karnataka and Andhra Pradesh. Draw area rectangles. Real-time coordinate capture with multiple geological layer overlays.

Random Forest AI Model

Trained on NGCM geochemical data (68 elements, 10,004 samples) joined with GSI lithology polygons. Achieves R² = 0.855 accuracy on test set.

Prospectivity Heatmaps

Dynamic density heatmaps generated from mineral occurrence registry. Red = High potential, Yellow = Medium, Green = Low. Toggle on/off.

Explainable AI (XAI)

Every prediction comes with a "Why this result?" breakdown — feature importance, geochemical anomalies, lithology matching, and proximity scores.

PDF Exploration Reports

Professional multi-page PDF reports with mineral inventory, concentration percentages, lat/lon/altitude, geological zone, and XAI interpretation.

MongoDB Cloud Storage

All predictions stored in MongoDB Atlas. Browse prediction history, compare results across locations, and track exploration campaigns over time.

Workflow

From Map Click
to Exploration Report

Five steps from location selection to professional PDF exploration report.

1
Select Location on Map

Click a point or draw an area on the interactive Leaflet map. Coordinates auto-fill.

2
Spatial KNN Geochemistry

System finds the nearest NGCM stream sediment sample and retrieves local background geochemistry.

3
AI Model Prediction

Random Forest Regressor predicts mineral probability score (0–100%) from 12 geospatial features.

4
Mineral Inventory

Element enrichment analysis (70th percentile thresholds) identifies likely minerals with concentration estimates.

5
Report Generation

Download a comprehensive PDF report with all findings, heatmap, and XAI explanation.

About the Project

Built for Hackathons,
Designed for Industry

GeoMiner AI was built to democratize mineral prospectivity analysis — a field previously requiring expensive GIS software and specialist knowledge.

By combining national geochemical databases, geological shapefiles, and machine learning, GeoMiner AI delivers exploration-grade predictions through a modern web interface accessible to geologists, researchers, and startups alike.
