Core Platform Capabilities

Spatial Target Discovery

Input coordinate names, specific latitudes, or simply click on the interactive Leaflet GIS map to pinpoint mineral targets.

Predictive ML Models

Evaluates coordinates against pre-trained Random Forest and XGBoost regressors, delivering mineral potential scores and likely commodities.

Explainable AI (XAI)

Understand predictions through element threshold enrichment analysis, distance-to-known-deposit decay indices, and global feature importance charts.

Geological Reference Data

  • NGCM Geochemical Database: Ingests 10,004 stream sediment records mapping 68 chemical elements and oxides in Karnataka & Andhra Pradesh.
  • Lithology Map Layer: Spatially intersects points against 3,599 shapefile polygons representing formations, standard rock types, and stratigraphy.
  • Mineral Occurrence Registry: Integrates coordinates for 57 known mineral mineralization spots and quarries for distance calculations.

Enterprise Technology Stack

  • Node.js & Express: Fast backend REST API controller serving static pages and managing predictions DB logic.
  • MongoDB (Mongoose): Stores prediction history logs and geocoding settings with offline file-based fallback support.
  • Interactive Maps & Charts: Dynamic mapping rendered via Leaflet.js (OSM tiles) and analytics drawn using Chart.js.
